# PKGBUILDs for [fastly/cli](https://github.com/fastly/cli)

## Updating Version

```sh
$ ./update-cli.sh X.Y.Z
```

For example, for [v0.26.3](https://github.com/fastly/cli/releases/tag/v0.26.3):

```sh
$ ./update-cli.sh 0.26.3
```
